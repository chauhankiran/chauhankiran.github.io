Python for Programmers

As the name suggests, this article is for the programmers who know the other programming language(s) and want to try or work with Python.

You need Python on your computer. Go ahead and open the official website. You need to install the latest version of Python (at this moment it's v3.9.x). Python has .py extension. 

print("hello, world")

We just wrote the hello, world program in python. The print() function is used to print the text/string on the terminal. It also adds \n at the end.

To run the Python program, this command is used (assuming that the name of the file is hello.py).

$ python hello.py

Python does not require a semicolon to complete the statement. It is helpful when you want to write multiple statements in single lines and want to separate them. 

print("hello, world"); print("hello, world")

Instead of double quotes for the string, you can also use single quotes. So, the following code is identical to the above code. 

print('hello, world')

I would suggest you to use double quotes unless the string has double quotes. Even in that scenario, you can use escape sequence characters such as \n. 

print("hello \" world")

This returns the following output. 

$ python program.py
hello " world

To write a comment, # is used. 

# hello, world program
print("hello, world")

String concatenation is done via + operator. 

print("hello, " + "world")

Python will throw an error when you try to concatenate string with number. 

print("hello, " + 4)

Running the above program will throws the following error. 

$ python program.py
Traceback (most recent call last):
  File "E:\Python\program.py", line 1, in 
    print("hello, " + 4)
TypeError: can only concatenate str (not "int") to str

In this case, we need to convert the number into string using str() function to run the string concatenation operation. 

print("hello, " + str(4))

In similar way, you need to use int() function to convert the string into number. 

print(int("3") + 23)

Python is dynamic programming language. You don't have to worry about the types while defining a variable. Just define a variable with its initial value and you're good to go. For example, here I have defined a string and integer variable. 

name = "Bruce"
age = 28

To print the values of these variables using print() function, we need to cast age variable into string as its number and will throw the above error if we try to concatenate it with string. 

name = "Bruce"
age = 28

print("Hello " + name + "! You're " + str(age) + " years old.")

As expected, it'll print the following output. 

$ python program.py
Hello Bruce! You're 28 years old.

The string concatenation is cumbersome. You can use .format() method to simplify the print operation with variables. 

name = "Bruce"
age = 28

print("Hello {}! You're {} years old.".format(name, age))

This will print the same identical output as above. The .format() method replace the {} with variable mentioned in same sequence i.e. first {} with name and second {} with age.

Or you can even use f-strings as follow. 

name = "Bruce"
age = 28

print(f"Hello {name}! You're {age} years old.")

Again, this will print the same identical output as above two. But, this is lot cleaner. You just need to prefix the string with f.

To get something from the user input() function is used. 

name = input("Enter the name: ")

print(f"Hello, {name}")

input() function will wait for the user to type something and accept the characters until Enter button is pressed. 

$ python program.py
Enter the name: Bruce
Hello, Bruce

input() function return the string. So, if you're planning to do some mathematical calculation on the entered input, you first need to convert the string into number using int() function. 

price = int(input("Enter the price of product: "))

payable = price - 4.5

print(f"You need to pay {payable} after discount.")

If we don't wrap the input() function with int(), the addition operation will throw the error. 

python program.py
Enter the price of product: 12
Traceback (most recent call last):
  File "E:\Python\program.py", line 3, in 
    payable = price - 4.5
TypeError: unsupported operand type(s) for -: 'str' and 'float'

Python supports the arithmetic operators such as - +, -, *, /, %. They're usual and you'll find these in almost all programming languages. Python also has ** or power operator and // or integer division operator that truncates the value after decimal point. 

print(3 + 12.4)     # 15.4
print(45 - 29)      # 16
print(34 * 17)      # 578
print(36 / 17)      # 2.1176470588235294
print(3 ** 4)       # 81
print(36 // 17)     # 2

Interesting enough, * also works with string and do the string multiplication with given number. 

print("hello " * 6)

This program runs without an error and returns the following output.

python program.py
hello hello hello hello hello hello

Python play nicely with relational operators such as - >, <, >=, <=, ==, and !=. When used, they'll return the Boolean output in terms of True or False. 

print(3 > 4)   # False
print(3 < 4)   # True
print(2 >= 1)  # True
print(6 <= 3)  # False
print(3 == 3)  # True
print(3 != 3)  # False

These False and True are keywords used to present the truth values..

Talking about Boolean, Python has support for Boolean operators such as and, or, and not. 

print(True and False)  # False
print(True or False)   # True
print(not False)       # True

As like other programming languages, the order of operators matters and whenever in confusion, always use parenthesis.

Python is one of the rare language in which indentation has meaning. For example, following program throws an error just because second line is indented. 

print("hello, world")
  print("hello, world")

Python throws an error because, indentation is used to create a block unlike curly braces in other programming languages.

For example, to write the if condition, following syntax is used.

num = 4

if num == 4:
  print("The magic number")

Here the indentation creates the body of if condition. Also, : is used after the condition to mark the end of condition expression.

In similar way else body must be written. 

num = 4

if num == 5:
  print("The magic number")
else:
  print("Not an ordinary number")

Write a statement without indent and you're in normal execution flow than this if-else condition. 

num = 4

if num == 5:
  print("The magic number")
else:
  print("Not an ordinary number")

print("And this print always")

To write else if condition elif keyword is used with condition. 

num = -3

if num > 0:
  print("The number is positive.")
elif num < 0:
  print("The number is negative.")
else:
  print("The number is zero")

Nested condition can be written in Python (that make program less readable).

while is used to write the looping functionality in Python. Again, indentation is used to create the body of while condition. 

i = 0

while(i < 6):
  print(i)
  i = i + 1

The while loop prints number from 0...5 and end as condition becomes false.

for...in loop can be written using range() function. 

for i in range(6):
  print(i)

This will print the number from 0...5, excluding the number mentioned in range() function.

We can even mention from where to start and end in range() function as follows. 

for i in range(1, 6):
  print(i)

This loop will print the number from 1...5. Finally, you can mentioned the step through within range() function. 

for i in range(1, 10, 2):
  print(i)

This loop will print the number from 1, 3, 5, ...9.

range() function is useful when you want to loop through the string characters. 

for char in "hello, world":
  print(char)

This will print each characters from the mentioned string.

I personally find for...in useful while working with List (e.g. Array in other programming language(s)).

Let's talk about the List first and then we'll review the for...in loop with List.

Box bracket is used to define the List. Following define the lists l1 and l2. 

l1 = []
l2 = ["apple", "banana", "orange"]

.append() method add an element to list at the end. 

l1.append("one")

After this operation, value of li becomes ['one']. To add an element at the given location, .insert() method is used. 

l2.insert(2, "grapes")

grapes will be added at the location 2 after banana as location starts from 0.

Also, there is an extend method that will extend the one list with another. 

l1 = ["one"]
l2 = ["apple", "banana", "orange"]

l1.extend(l2)

print(l1)

The list l1 is now extended and return following output. 

$ python program.py
['one', 'apple', 'banana', 'orange']

To remove an element from the List, .pop() method is used. It'll remove the last element from the list. 

l = ["apple", "banana", "orange"]

print(l.pop())

This will remove the "orange" from the list. If .pop() is mentioned with index, then it'll remove the element from the particular index. 

l = ["apple", "banana", "orange"]

print(l.pop(0))

This will remove the apple.

You can even remove the element by specifying the text to .remove() method. 

l = ["apple", "banana", "orange"]

l.remove("banana")

This removes the banana from the list.

You can use del keyword to delete the element from the specifies index. 
l = ["apple", "banana", "orange"]

del l[0]

This remove the apple from the list. Finally, with del keyword, you can even delete the complete list. 

l = ["apple", "banana", "orange"]

del l

To loop through elements from list, for...in is used. 

fruits = ["apple", "banana", "orange"]

for fruit in fruits:
  print(fruit)

def keyword is used to define a function Python. 

def hi():
  print("hello, world")

hi()

Last line call the hi() function and prints the text. 

def add(a, b):
  return a + b

print(add(3, 4))

This add() function accept two parameters and return the sum of those.

Function has support for default parameters value. 

def add(a=4, b=4):
  return a + b

print(add(3, 4))
print(add())

In above program, the last line is used to call the function without arguments and in this case it run the function with default parameters.

While calling the function, you can use the name of parameters to call the function. In this way, you don't have to worry about the sequence of argument. 

def add(a, b):
  return a + b

print(add(b=4, a=3))

Here I reverse the order of argument and it still work because we have passed the name.

Python function supports the variable arguments. 

def marksheet(name, *marks):
    total = 0
    for mark in marks:
        total = total + mark

    print(f"Hi, {name}. Your total marks is {total}.")

marksheet("Bruce", 43, 42, 47, 39)
marksheet("Bruce", 47, 43, 44)

The first argument is points to name and then remaining arguments points to marks. Variable arguments parameter should be define with * e.g. *marks. Running this program will print the following output. 

$ python program.py
Hi, Bruce. Your total marks is 171.
Hi, Bruce. Your total marks is 134.

Functions are great ways to modularize the application into manageable and re-usable code. You can go further and modularise the application in modules also. Before we go further and create our own modules, let's become familiar with existing/standard module.

To use standard module, import statement is used in following way. 

import sys

This import the sys module and all the functions or constants from the sys modules can be used by prefixing sys. as follow. 

sys.copyright
This return the copyright statement.

You can remove this prefix sys. by importing the module using from...import statement. 

from sys import *

print(copyright)

Here * means everything. Everything from sys module is imported. We can even import the specific item from the module by mentioned the name of it. 

from sys import copyright

print(copyright)

This program now just import the copyright constant.

We can even import specific items from the given module by using from...import statement.

We can create our own module just by creating a file new file and include it. Create a file with name module.py and write the following code. 

def hi():
  print("hello, world")

Now, in our program hello.py, to import this module, write - 

import module

module.hi();

Or we can use the from...import statement as 

from module import hi

hi()
When you import something for the first time, Python execute the module completely. For example, if module.py programs has 
def hi():
  print("hello, world")

print("This should not be printed.")

In our program, even though we're just calling hi() function from the module.py, it'll print the last line. 

$ python program.py
This should not be printed.
hello, world

If we don't want to print that last line, it should be places within condition as follow. 

def hi():
  print("hello, world")

if __name__ == "__main__":
  print("This should not be printed.")

This __name__ == "__main__" is only executed when the module.py program is executing standalone not as import. Hence, the print() will not run.

Apart from List, you have support for Tuple. Tuple is similar to List except it is immutable. You can change in any possible way. Normal parenthesis are used to define the tuple although parenthesis are optional. For example, 

fruits = ("apple", "cherry", "banana")

You can confirm the type of tuple using type function as, 

fruits = ("apple", "cherry", "banana")

print(type(fruits))

This will return the <class 'tuple'>.

As mentioned, parenthesis are optional. Hence, if you define a tuple within single element as, 

fruits = ("apple")

Printing the type of should be str. To define an actual tuple, we need to write , at the end as - 

fruits = ("apple", )

Now, this is defined as tuple. While we can't modifies the tuple, the only thing we can do is, access the particular element using index and get the length of tuple using len() method. 

fruits = ("apple", "banana", "cherry")

print(fruits[1])
print(len(fruits))

Here for...in can be used to loop through. 

fruits = ("apple", "banana", "cherry")

for fruit in fruits:
    print(fruit)

Python has support for Dictionary (Object in other programming language e.g. key/value pair). Curly brackets are used to define the dictionary. 

person = {
    "name": "Bruce",
    "age": 28
}

extraInfo = {}

To access the element from the dictionary, [] is used. 

person = {
    "name": "Bruce",
    "age": 28
}

extraInfo = {}

print(person["name"])

To remove item from the dictionary, .pop(), .popitem(), and del keyword can be used. 

person = {
    "name": "Bruce",
    "age": 28
}

person.pop("age")
print(person)

The above code remove the age from the person dictionary. 

person = {
    "name": "Bruce",
    "age": 28
}

person.popitem()
print(person)

.popitem() remove the last item from the dictionary. Alternatively, you can write - 

person = {
    "name": "Bruce",
    "age": 28
}

del person["age"]
print(person)

To add something, just define the key with box notation and assign the value. 

person = {
    "name": "Bruce",
    "age": 28
}

person["city"] = "Gotham"

print(person)

for...in used to loop through the dictionary items. 

person = {
    "name": "Bruce",
    "age": 28
}

for detail in person:
    print(detail)

This prints the key e.g. name and age. To print the value, we need to use box notation. 

person = {
    "name": "Bruce",
    "age": 28
}

for detail in person:
    print(person[detail])

Or we can use .values() method. 

person = {
    "name": "Bruce",
    "age": 28
}

for detail in person.values():
    print(detail)

len() function is used to get the length of dictionary. 

person = {
    "name": "Bruce",
    "age": 28
}

print(len(person))
