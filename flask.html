# Flask We're going to learn the Flask, a framework for building web applications in Python. Bookmark the official website as we're going to visit this website a lot. Getting up and running the web application with Flask is as simple as running few commands in terminal and few lines of Python code. Create a folder with name hello-flask and the go inside. mkdir hello-flask cd hello-flask Create a virtual environment for this project to isolate the project and it's dependencies. python3 -m venv .venv Activate the virtual environment using following command. . .venv/bin/activate After running this command, you should see a different in command prompt. Let's now install the Flask. Running the following command will do the same. pip3 install Flask This command might take few second to install Flask and related dependencies. Once done, open the project folder in your favorite text editor. Create a new file with name app.py. touch app.py Open this file in text-editor and write the following code. from flask import Flask app = Flask(__name__) Go ahead and create a root route that return hello, world text when visited in web browser. For this, we need to write following code. from flask import Flask app = Flask(__name__) @app.route("/") def index(): return "hello, world" Our application is ready to be served. Run the following command to run the application. flask run You should see that the application is running at http://localhost:5000. Visit this URL in web-browser and you should see the hello, world text as that's what we have written in code. We can update the return text by making it as header as follow. from flask import Flask app = Flask(__name__) @app.route("/") def index(): return "

hello, world

" If you reload the URL in browser, you wouldn't see the updated result. We need to restart the server to see the updated output. Go ahead and terminal the running application by pressing ctrl+c twice and then re-run the flask run application. Then visiting the URL should returns the updated result. Re-starting the server again and again is not good idea in development. Luckily, flask resolve this problem by running the flask run command using --debug option as follow. flask run --debug Let's test this by modifying the response from HTML header to JSON. To return JSON, we need to use jsonify() method from Flask and need to return JSON object as follow. from flask import Flask, jsonify app = Flask(__name__) @app.route("/") def index(): return jsonify({ "message": "hello, world" })
