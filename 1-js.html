Node & NPM

This is one of the fundamental software you need on your computer. We are using Windows operating system – in our organization so that this documentation contains the installation steps for given operating system only.

The installation of Node on Windows is quite simple. You just need to download the LTS .msi executable file of Node from it’s official website. After that just double click on downloaded .msi and follow the steps suggested by installer. At the end of this installation process you will have Node and NPM on your computer! You can confirm the installation by running following command in Windows Command Prompt.

$ node --version
v8.12.0
$ npm --version
6.4.1

That’s it! These are the only software needed for first week. You are good to go. But, before that do following things:

  1. You can have different versions than mentioned above. But, you must have node version greater than 8 and NPM version greater than 5.3.
  2. If you are facing any problem while installing any software on your computer, please communicate with or me.
  3. Finally, please send the confirmation email to me that you have successfully installed this software with versions and operating system.
